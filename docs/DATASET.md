# Dataset (Synthetic) — Directed Speech (RU)

Датасет генерируется скриптом `scripts/generate_data_v3.py`.

## Labels
- `label=1` — реплика адресована ассистенту (команда/вопрос/управление диалогом)
- `label=0` — реплика не адресована ассистенту (разговор с людьми, фон, small talk)

## Columns
- `text` — текст (часто после ASR-подобной порчи)
- `label` — 0/1
- `group_id` — идентификатор шаблонной группы для предотвращения leakage при split

## Noise model (приближение ASR)
Применяются вероятностные преобразования:
- добавление заполнителей (“эм”, “ну”, …)
- удаление пунктуации/запятых
- случайные символьные ошибки (удаление/замена)
- частичные фразы (обрыв в конце/начале)
- случайное приведение к нижнему регистру

## Post-noise validation for positives
После шума позитивные примеры валидируются (чтобы не остались “пустые” команды из одних заполнителей).

## Known limitations
- Синтетика не покрывает весь спектр разговорной речи/диалектов.
- Качество на реальном ASR зависит от домена и может потребовать донастройки порога `threshold`.
